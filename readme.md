# Portfolio Website

This is a personal portfolio website hosted on GitHub Pages. I created this project as part of my Web Development Internship at Elevate Labs (Ministry of MSME, Govt. of India).

The main objective of this task (Task 5) was to learn how to deploy a static website using GitHub Pages and manage version control with Git.

## Live Demo

You can view the live website here: [https://vrindamodi.github.io](https://vrindamodi.github.io)

## Technologies Used

- **HTML5**: For structure and layout.
- **CSS3**: For styling and responsive design.
- **JavaScript**: For simple interactivity.

## Features

- Responsive design that works on mobile and desktop.
- Clean, minimal user interface.
- Sections for About, Skills, and Projects.

## Project Structure

- `index.html`: Main entry point.
- `style.css`: Stylesheet.
- `script.js`: Script file.

## Setup

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Vrindamodi/Vrindamodi.github.io.git
   ```
2. Open `index.html` in your web browser.
